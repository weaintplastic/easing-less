easing.less
===========

A css3 easing mixin library based on less. Please find a demo here http://weaintplastic.github.com/easing.less/


## CSS usage

```html
	<div class="ease-in-out-quart"></div>
```


## LESS usage

```less
	#element-to-ease{
		.ease-in-out-quart(.5s, all);
	}
```
